All the parts of the codes can be run on Google Colab. 
To run the code, you have to put our directory at the following root directory in your Google Drive:
'/content/gdrive/MyDrive'


Code organizations:
We have created different file for different tasks.
	- Analysis of Queries and Documents.ipynb --> contains the first part of the Assignment
	- Basic Search --> contains the second part of the Assignment
	- Query expansion.ipynb --> contains the analysis to expand adhoc queries.
	- Query reduction.ipynb --> contains the analysis to reduce descriptive and narrative queries.
	- Re-ranking by date.ipynb --> contains the functions to re-rank the documents retived using the date.
	- BERT re-ranking.ipynb --> contains the code to re-rank the documents retived using BERT.
